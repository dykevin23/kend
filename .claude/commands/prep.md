readme 폴더 하위의 md 파일들을 모두 읽고 프로젝트 컨텍스트를 파악해줘.

폴더 구조:
- readme/core/ - 시스템 기반 reference 문서 (architecture, auth, database 등)
- readme/active/ - 진행 중인 plan/todo 문서
- readme/todo/ - 아직 시작 전 plan 문서
- readme/archive/ - 완료/보류된 문서
- readme/changelog-kend.md - KEND 웹앱 변경 이력
- readme/changelog-seller.md - KEND-SELLER 변경 이력
- readme/changelog-native.md - KEND-NATIVE 변경 이력

파일을 읽은 후 현재 프로젝트 상황을 간략히 요약해줘.
